About

Find My Face is a Portfolio Project for Holberton School developed by Jose Fabian Rosa, Jeffrey Martinez and Jared Beguellin. It was created thanks to the initiative of Jared Beguellin, in order to use the Python library "face_recognition" to implement methods where you can make a facial recognition of a person in a group image. We made a graphical user interface, where the user on one side must upload the image of the desired person and on the other side the group image where they want to find the person from the first image. To achieve the desired result we use several languages and technologies: Python (Backend), HTML / CSS and JavaScript (Frontend) and Apache2 (Server). For future ideas, we want to implement that the user can upload a file with up to 10 group images to find the desired face.
